iMyApps 服務

iMyApps 是一個記錄您購買的 iPhone/iPad 軟體的服務。
透過一個客戶端程式讀取已下載的 App，並將App資訊儲存到雲端。
(下載回來的App是 *.ipa 檔案，位於iTunes Media資料夾下)
之後隨時隨地都可以從 iMyApps 服務頁面上查詢下載過的App記錄！

開始使用 - Step by step

  1. 註冊 iMyApps 帳號
  2. 至您註冊的信箱收取認證信件，並點擊信件中的網址
  3. 下載 iMyApps客戶端
  4. 執行 iMyApps 客戶端程式(可於每次電腦開機時執行客戶端程式進行同步)
  5. 登入 並開始使用各項功能

功能說明

  1. 列表：列出您記錄的App，並可幫 App 添加備忘說明
  2. 搜尋：搜尋您記錄的App
  3. 統計：統計您下載的App數量及花費統計(美金)
  4. 個人檔案：修改您的個人相關資訊
*注意*
若您下載 App 後，在進行同步前即將 App 移除，且移除 App 時未保留 IPA 檔案， 您之後將無法查詢此 App 的相關記錄。

統計資訊

iMyApps 客戶端

iMyApps 客戶端是用於將下載的App(*.ipa)資訊同步到雲端的程式。
  1. 支援 Microsoft Windows
  2. 需 Java Version 6 (JRE 1.6.0) 以上版本 (下載)

iMyApps 客戶端程式設定檔的路徑位置

  Vista 及 Windows7
    C:\Users\Username\AppData\Local\iMyApps\config.xml
  Windows XP
    C:\Documents and Settings\Username\Application Data\iMyApps\config.xml

設定檔格式：



設定檔中可以設定的參數：

  1. apppath: 存放 IPA 檔的資料夾路徑
  2. delay: 程式啟動後，延遲幾秒後再進行連線動作
  3. buyer: 針對特定的購買帳號進行記錄，若有多個可用 ',' 隔開！(預設為所有讀取到的IPA檔都會記錄)

預設的 iTunes App 資料夾位置：

C:\Users\Username\Music\iTunes\iTunes Media\Mobile Applications
若您不是使用預設的 iTunes Media資料夾，需要修改相關路徑，請於設定檔中加入
